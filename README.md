# jopbrainlib-notebook
A simple Jupyter interface to use [JopBrainLib](https://github.com/jeroenkroesen/jopbrainlib).  
  
  
For more information refer to [JopBrainLib](https://github.com/jeroenkroesen/jopbrainlib).  
  
  
## Installation (using uv)
Clone this repository:  
```sh
git clone https://github.com/jeroenkroesen/jopbrainlib_notebook
```
  
Go into the repo:
```sh
cd jopbrainlib_notebook
```  
  
Create virtual env and install dependencies:
```sh
uv sync
```
  
***  
  
  
## Usage
Activate your virtual env:
```sh
source .venv/bin/activate
```
  
Start Jupyter Lab:
```sh
jupyter lab
```
